# ReactPushNotification
 
 
## Description
This library allows us to use html5 NotificationAPI in Reactjs. 
 
## Demo
The following is the most simple usecase.

```jsx
Notif('Title Massage');
```
OR
```jsx
Notif('Title Massage', {
   body: 'Body Massage',
   icon: "http://cdn.sstatic.net/stackexchange/img/logos/so/so-icon.png"
});
```
